# Lista de Exercícios - STRUCT

Projeto desenvolvido para a disciplina de Estrutura de Dados.

## Objetivo

Implementar Tipo Abstrato de Dados usando `struct` em C#, com dados e operações para os seguintes contextos:

1. Jogador de Futebol
2. Equipe de Esports
3. Produto
4. Professor

## Estrutura do projeto

```txt
ListaStruct/
├── Program.cs
├── JogadorFutebol.cs
├── EquipeEsports.cs
├── Produto.cs
├── Professor.cs
└── ListaStruct.csproj
```

## Como executar

No terminal, dentro da pasta do projeto:

```bash
dotnet run
```
